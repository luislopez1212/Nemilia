Nemilia AI Roadmap
v1.07 baseline | Updated February 2026 

Q2 2026 — Output & Voice
Document Export

DOCX output with canvas.toBlob() for embedded charts (resolves the image issue)

TXT and Markdown export

PDF upgrade: embedded SVG charts, better typography, optional TOC

Voice Input

Web Speech API as default — browser-native, free, zero data leaves the machine

Groq Whisper API as opt-in — user's own key, high accuracy, multilingual

Keyboard shortcut: Ctrl+Shift+V to activate

Quick Wins

Import/export individual agents without exporting full workspace

Cost tracking: total tokens and spend per provider

Agent performance log: track average quality scores per agent over time

Q3 2026 — Intelligence & Reach
Multi-Modal VISION Upgrade

Accept image uploads alongside prompts for vision-capable models (GPT-4o, Gemini Flash, Claude Sonnet)

Use case: paste a screenshot, let agents analyze it

Audio file upload → Whisper transcription → downstream agent pipeline (meeting notes, interviews, etc.)

RAG Improvements

Full webpage extraction via NEXUS (not just search snippets)

Persistent summarized memory across sessions so agents retain long-term context without re-uploading docs

Workflow Portability

Export any workflow as a shareable JSON bundle (agents + prompts + settings, no keys)

One-click import on any Nemilia instance

Generates a standalone HTML report from any workflow run — shareable with anyone, no app required
