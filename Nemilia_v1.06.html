About Nemilia AI
v1.06 · Human-in-the-loop · Semantic vector RAG · Drag-and-drop workflows
Nemilia (Neh-mee-lee-ah) — from the Nahuatl verb meaning to think, to remember, or to imagine. Chosen because thinking and remembering are exactly what an LLM does when leveraging persistent memory and cross-run context.

A single HTML file that runs entirely in your browser. No server, no install, no account. Bring your own API keys.

Providers: Groq · OpenRouter · Gemini · Anthropic · OpenAI · Mistral · DeepSeek · Kimi (Moonshot) · xAI · Ollama · LM Studio · Jan · Llamafile
Search: Brave Search · Serper (Google) · Tavily AI · SearXNG (self-hosted)
Storage: File System API (Chrome/Edge) · IndexedDB (Firefox/Safari) · localStorage fallback
RAG: Semantic vector search (Transformers.js) + BM25 hybrid · PDF · DOCX · TXT · MD · CSV
Pipeline: DAG staggered execution · Human-in-the-loop (HITL) checkpoints · Drag-and-drop workflow builder · Role-based tool access · SLM prompt compression · SVG flow visualisation
Security: Hardened XSS esc() · CSP meta tag · AES-256-GCM · 200k PBKDF2 · stream error fallback · import validation · color sanitisation
